This project is based on the CDLI corpus of Proto-Cuneiform texts from the Uruk IV and III periods. It also hosts a signlist which is based on several sources: the CDLI proto-cuneiform signs GitHub repository [https://github.com/cdli-gh/proto-cuneiform_signs]; Anshuman Pandey's Revised proposal to encode Proto-Cuneiform in Unicode [https://www.unicode.org/L2/L2023/23190-proto-cuneiform.pdf]; and revisions and additions by Steve Tinney.
